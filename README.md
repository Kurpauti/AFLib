# Atelier Fazzy Library(AFLib)  

AFLib(Atelier Fazzy Library)は**DirectXを使ったC++向けのゲーム制作用ライブラリ**です。  
現在は自分用にのんびり開発していますが、進捗を定期的にGithubに公開していく予定です。

## このライブラリの概要
- C++用のゲーム制作向けライブラリ
- DirectXベース
- ゲーム開発において個人的に「あったら便利だな」を寄せ集めたもの

## 開発状況
現在は**開発途中**です。  
設計も実装も頻繁に変わります。
